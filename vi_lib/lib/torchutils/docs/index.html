TorchUtils¶

This package provides some utility functions based on PyTorch.

GitHub - Documentation

You might also be interested in checking out

Inferno, ProbTorch, Pyro, Ignite

Installation¶

Installing from git¶

Editable mode:

git clone https://github.com/fKunstner/torchutils.git torchutils
cd torchutils
pip install -e .

Direct install:

pip install git+https://github.com/fKunstner/torchutils.git

Installing from PyPI¶

Currently disabled

pip install fk-torchutils

Modules¶

  • torchutils.low_rank - Fast Operations for Low-Rank + Diagonal Matrices
  • torchutils.distributions - Low-Rank + Diagonal Multivariate Gaussian
  • torchutils.models - (very) Simple models instantiation helpers
  • torchutils.fastpca - Fast Randomized PCA
